# CALZADO J&R - Backend

Backend desarrollado con FastAPI para el sistema de gestión de CALZADO J&R.

## Tecnologías

- Python 3.12+
- FastAPI
- SQLAlchemy 2.0
- PostgreSQL
- JWT Authentication
- uv (gestor de dependencias)

## Instalación

```bash
# Instalar dependencias
uv sync

# Ejecutar servidor de desarrollo
uv run uvicorn app.main:app --reload
```

## Documentación API

Una vez corriendo, visita: http://localhost:8000/docs
